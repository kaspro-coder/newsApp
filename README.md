# StayTuned

**All your news, right at your fingertips.**

<img width="100" height="400" alt="logo_stayTuned" src="https://github.com/user-attachments/assets/0090e84b-5bca-4ae4-9cd6-80132d2e4495" />

StayTuned is a mobile news application built with **React Native** and **Expo**, designed to deliver a smooth and enjoyable user experience. It enables users to create an account, customize their profile with a photo, and (coming soon) access a personalized news feed.

## Features

- **Account Creation**: Full user registration including first name, last name, email, and password.
- **Secure Authentication**: User login and session management powered by Firebase Authentication.
- **Profile Picture**: Users can select and upload a profile picture from their photo library.
- **Database**: User data is stored in a Firestore database.
- **File Storage**: Profile pictures are stored securely using Firebase Storage.

## Technologies Used

- **Framework**: React Native (with Expo)
- **Backend (BaaS)**: Firebase
- **Authentication**: Firebase Authentication
- **Database**: Cloud Firestore
- **Storage**: Firebase Storage
- **Navigation**: Expo Router
- **Permissions & Media**: `expo-image-picker`, `expo-file-system`
- **Local Storage**: `@react-native-async-storage/async-storage`
